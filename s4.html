List Slicing : We use list slicing to specifiy the elements we need in a list by using a colon ( : ) as the following :

1- List[ index-a : index-b ] => From index-a to index-b.

( index-b is bigger than index-a )

2- List[ index-a : index-b ] => Empty list.

( index-a is bigger than index-b )

3- List[ : index-a ] => From the start of the list to index-a.

4- List[ index-a : ] => From index-a to the end of the list.

5- List[ : ] => All the list.

6- List[ index-a : index-b : index-c ] => From index-a to index-b with a step equal to index-c.

( index-c is a positive value )

7- List[ index-b : index-a : index-c ] => From index-b to index-a with a step equal to index-c.

( index-c is a negative value )

Sets : They are created with semi colons ( ; ) like dictionaries, but they have normal values and not keys and values. They also automatically delete any repeated values in them.

Some Methods and Functions Related To Sets :

1- union() => Combine two sets.

2- intersection() => Checks the similarities between two sets.

3- set() => Converts a list to a set.

4- list() => Converts a set to a list.

The del keyword is used to remove a variable or any item from the program memory.

Lambda Function : Similar to normal functions, except they don't require the def keyword ( We use the lambda keyword instead ) and are written in one line.
