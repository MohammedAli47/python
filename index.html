Syntax : An arrangement of letters and words to perform a certain action.

Variables : places to store data.


Rules to write variables :
  1. 1-First letter : No numbers.
  2. 2-The whole word : No special characters (except the underscore _)
  3. 3-Variables cannot be named as keywords.

Ex: _name1 / Fullname / Tamer135Ahmed


Data types :
  1. 1-Integer (int) : 5 / 75 / -980 ...
  2. 2-Float (float) : 4.23 / 70.5 / 5.0
  3. 3-String (str) : "Ahmed" / "First_Name" / "Hello world!" / "25"
  4. 4-Boolean (bool) : False ( 0 ) / True ( Any value except 0 )

Comments:

( # ) at the beginning for one line comment.

(""" text """) for muultiple lines of comments.



Input : Use the input() function.

Output : Use the print() function.

Note!! print("name") isn't the same as print(name).


Operators:

1-Arithmetic :

a + b => (Addition)

a - b => (Subtraction)

a * b => (Multiplication)

a / b => (Division)

a ** b => (Exponentiation)

a % b => (Modulus)


2-Assignment :

a += b => a = a + b

a -= b => a = a - b

a *= b => a = a * b

a /= b => a = a / b

a **= b => a = a ** b

a %= b => a = a % b

3-Comparison :

( > ) => Greater than

( >= ) => Greater than or equal

( < ) => Less than

( <= ) => Less than or equal

( == ) => Equality check

( != ) => Non-equality check


4-Logical :
Comparison Statement is true if: Statement is false if:
and both is true both is false
or at least one is true at least one is false
not it is false it is true
if statement :

if( first statement ):

(Indentation) Code

elif( second statement ):

(Indentation) Code

elif( third statement ):

(Indentation) Code

elif( fourth statement ):

(Indentation) Code

else:

(Indentation) Code

The computer checks the first statement, if it's false, it goes to the second statement to check it and so on... .You can write as many or as few elif statements as you want. (You can just write if and else without elif).


String Concatination :

" text " + " other text " = " text other text "

They get combined together.

String Repetition :

x = "Ali" * 3 = AliAliAli
